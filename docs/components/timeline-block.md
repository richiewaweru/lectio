# TimelineBlock

## Purpose
Turn chronology into a readable instructional spine so learners can follow cause, sequence, and turning points over time.

## Best Uses
- History and discovery stories
- Science timelines with key turning points
- Any lesson where sequence is the meaning, not just background detail

## Inputs
- `title`
- `intro`
- `events`
- `closing_takeaway`

## Behaviours
- `static`
- `timeline-scrubber`

## Print
Flattens into a readable vertical event list with the chronology preserved.
