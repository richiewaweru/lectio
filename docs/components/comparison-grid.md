# ComparisonGrid

## Purpose
Keep two to four concepts in view at the same time so learners can compare them by explicit criteria instead of buried prose.

## Best Uses
- Binary contrasts such as alkanes vs alkenes
- Three-way or four-way classifications
- Lessons where the correct answer depends on noticing one structural distinction

## Inputs
- `title`
- `intro`
- `columns`
- `rows`
- `apply_prompt`

## Behaviours
- `static`

## Print
Renders as a static comparison table with the same criteria and values kept visible.
